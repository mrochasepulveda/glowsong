Riff UX strategy defined in March 2026. Key decisions:
- Core loop: PULSE → DISCOVER → COMMIT → SHARE → PULSE
- Navigation: TONIGHT (time-aware) | FOR YOU (daily drops) | MY SET (identity)
- Key differentiator: urgency anticipation — FOMO calculado con data real
- Visual: "club a las 2am" — spotlight-not-flood, time-based materiality
- Interactions: triple gesture (up=VOY, right=interesa, left=skip), daily drop system, boarding pass tickets
- Background darker to #06060E, dramatic typography ratio 3:1
- Social layer: crews, social proof on cards, afterglow post-event
